dbstatus

------------

### 功能介绍
- 输出mysql的qps，conn，innodb rows reads delete 等信心，每秒输出一次

### 使用举例
    -P=3306: port
    -h="127.0.0.1": host
    -p="test": password
    -u="test": user

    ./dbstatus -u=monitor -p=w123456 -h=127.0.0.1 -P=3306
